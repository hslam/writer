# writer
Package writer implements batch writing for an io.Writer object with the concurrency.

## Feature
* Shared Buffer
* Auto Batch Writing

## Get started

### Install
```
go get github.com/hslam/writer
```
### Import
```
import "github.com/hslam/writer"
```
